Отслеживание кликов 

curl -X POST https://linkoffer.ru/api/clicks 
-H "Authorization: Bearer your_api_key" 
-H "Content-Type: application/json"
-d '{
  "pid":"1", // ID сгенерированной партнёрской ссылки (Можно получить в разделе "Ссылки"). Должен совпадать с параметром pid в ссылке
  "project": "1", // ID проекта (Можно получить при просмотре проекта, нажав на проекте кнопку "Просмотр")
  "partner": "1", // ID партнёра (Указывается автоматически при генерации партнёрской ссылки)
  "referrer": "1", // ID платформы партнёра (Откуда пришёл пользователь)
}'

Отслеживание конверсий

Код для страницы успешного завершения:

curl -X POST 'https://linkoffer.ru/api/conversions' \
-H "Authorization: Bearer your_api_key" \
-H "Content-Type: application/json" \
-d '{
  "pid":"1", // ID сгенерированной партнёрской ссылки (Можно получить в разделе "Ссылки"). Должен совпадать с параметром pid в ссылке
  "project": "1", // ID проекта (Можно получить при просмотре проекта, нажав на проекте кнопку "Просмотр")
  "partner": "1", // ID партнёра (Указывается автоматически при генерации партнёрской ссылки)
  "referrer": "1", // ID платформы партнёра (Откуда пришёл пользователь)
  "amount": "1", // Цена конверсии (Опциональный параметр, должен быть не менее 85% от установленной цены)
  "details": "Конверсия" // Информация о конверсии. Пример: покупка, подписка и т.д. (Опциональный параметр)
}'

Примеры запросов 

POST api/clicks HTTP/1.1
Host: linkoffer.ru
Authorization: Bearer your_api_key
